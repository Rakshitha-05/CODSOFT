# Sales Prediction using Linear Regression

This project predicts product sales based on marketing spend.

## Workflow
✅ Exploratory Data Analysis  
✅ Feature Selection  
✅ Train-Test Split  
✅ Linear Regression Model  
✅ Evaluation using MSE and R2 Score  

## Dataset Columns Example
- TV
- Radio
- Newspaper
- Sales (Target)

## Results
R2 Score: Around 0.89 (varies depending on dataset)
